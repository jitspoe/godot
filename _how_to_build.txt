Open developer command prompt for vs 2019
scons p=windows tools=yes arch=x86 target=release_debug
(To try: arch=32, maybe that'll remove the double build?)

You can create a Visual Studio solution via SCons by running SCons with the vsproj=yes parameter, like this:
scons p=windows vsproj=yes

What I used:
scons p=windows tools=yes arch=x86 target=release_debug vsproj=yes

scons p=windows tools=yes arch=32 target=release_debug vsproj=yes -j10
